# Donkeycar UI 구현 프로젝트

## 프로젝트 개요
본 프로젝트는 Donkeycar에서 수집된 자율주행 데이터를 관리하고 학습을 실행할 수 있는 Windows 기반 UI 프로그램을 제작하는 프로젝트입니다.

Donkeycar는 카메라 기반 자율주행 오픈소스 플랫폼이며, 차량 주행 중 저장된 이미지와 조향 데이터를 이용하여 AI 모델을 학습시킵니다.

C# WinForms를 이용하여 데이터를 시각적으로 확인하고, Python 기반 학습 기능을 실행할 수 있는 관리 프로그램을 구현 예정입니다.

---

# 개발 목적

- 자율주행 데이터 구조 이해
- WinForms 기반 UI 프로그램 개발
- Python 기반 AI 학습 시스템 연동
- 이미지 및 JSON 데이터 처리 경험

# 주요 기능

## 1. 데이터 조회 기능
- catalog 파일을 읽어와 프레임별 이미지와 조향 데이터를 표시
- 이미지 출력
- angle / throttle 값 출력

## 2. 프레임 이동 기능
- TrackBar를 이용한 프레임 이동
- 현재 프레임 데이터 동기화

## 3. 리스트 선택 기능
- 특정 프레임 선택 
- 이미지 및 데이터 동기화 

## 4. 데이터 필터링 기능
- 특정 조건 데이터 검색 
- 불필요한 데이터 제거 

## 5. 데이터 삭제 기능
- catalog 데이터 삭제
- 이미지 파일 삭제

## 6. AI 학습 실행 기능
- Python 학습 명령 실행
- 학습 로그 출력
